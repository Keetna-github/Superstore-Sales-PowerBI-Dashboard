# Superstore Sales Dashboard - Power BI

## Project Overview
Interactive Power BI Dashboard analyzing 
Superstore Sales data with 9,994 records!!

## Pages
1. Overview - KPI Cards & Slicers
2. Sales Analysis - Regional & Monthly trends
3. Profit Analysis - Category & Product profit
4. Product Analysis - Treemap & Funnel chart
5. Customer Analysis - Top customers & Segments

## DAX Measures Created
- Total Sales
- Total Profit
- Total Orders
- Profit Margin %
- Average Order Value

## Tools Used
- Power BI Desktop
- DAX
- Power Query

## Dataset
- Source: Kaggle
- Records: 9,994
- Columns: 21
